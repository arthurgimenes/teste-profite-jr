## Getting Started


For install
* NodeJS -> https://nodejs.org/en/download/
* Git -> https://git-scm.com/downloads

Global dependencies required
* npm install -g bower

How to execute
* git clone https://github.com/armenio2/test-front-end-junior--profit.git
* npm install
* npm start

Open in your browser
http://localhost:3000
